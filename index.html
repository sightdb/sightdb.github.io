Quick Start Guide

Get started with SightDB in minutes using Docker:

docker pull koctep/sightdb
docker run -d -p 5984:5984 koctep/sightdb

Connect to SightDB using standard CouchDB client libraries:

curl http://localhost:5984
# Returns SightDB server information

Why Choose SightDB?

  • Combines PostgreSQL's reliability with CouchDB's flexible API
  • Optimized for high-performance in Kazoo environments
  • Easy integration with existing CouchDB applications
  • Built-in support for complex queries and transactions
  • Active development and community support
